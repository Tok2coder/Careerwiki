# CareerWiki 기술 컨설팅 리포트

> **작성일**: 2026-03-03
> **분석 범위**: 코드베이스 전체 (main 브랜치, commit f4ab654) + 프로덕션 사이트
> **분석 방법**: 3명 Analyst 병렬 분석 (Architecture / Product / Security & Ops)

---

## Executive Summary

CareerWiki는 150,010줄 TypeScript 코드로 구축된 **한국어 진로 정보 위키 + AI 추천 플랫폼**입니다. 6,945개 직업과 608개 전공 데이터를 통합하고, 6단계 AI 파이프라인으로 맞춤 추천을 제공합니다.

**종합 성숙도: Growth 단계** (MVP 단계를 넘어 실사용 가능하나, 운영 안정화와 보안 강화가 필요)

### 핵심 강점 3개
1. **AI 추천 파이프라인의 정교함** — 5축 프레임워크, 6단계 파이프라인(47개 파일), E2E 테스트 자동화까지 갖춘 경쟁력 있는 핵심 기능
2. **순환 의존성 0건의 건전한 아키텍처** — 150K줄 규모에서 routes → services → utils 단방향 레이어링이 잘 유지됨
3. **프로덕션 수준 세션 관리** — Opaque token + KV + 감사 로그 + Sliding window + 디바이스 관리까지 견고한 인증 체계

### 핵심 리스크 3개
1. **[Critical] API 레이트 리밋 완전 부재** — 모든 API가 무제한 호출 가능. AI 분석 API 악용 시 OpenAI 비용 폭주 직결
2. **[High] 테스트 커버리지 < 0.1%** — 150K줄에 테스트 1개(208줄). 리팩토링/기능 변경 시 회귀 버그 방어 전무
3. **[High] CORS 와일드카드 + CSP 미구현** — 모든 오리진에서 API 접근 허용 + XSS 방어선 부재

### 1.3 아키텍처 패턴 평가

**ISR 캐싱**: D1 캐시에서 CDN 캐싱으로 합리적 전환 완료. `page-cache.ts`에 죽은 코드(~40%) 잔존.

**데이터 통합 (Unified Profile)**: `api_data_json` + `user_contributed_json` + `admin_data_json` → `merged_profile_json` 3계층 구조. 개념 우수하나 병합 로직이 3개 파일에 분산.

**AI 파이프라인**: 6단계 흐름이 47개 파일로 명확 분리. 다만 오케스트레이터(`routes.ts`, 8,154줄)가 비대.

**Cloudflare Workers 적합성**: 대체로 적합. AI 파이프라인이 CPU 30초 한도에 근접(15-25초)하여 주의 필요.

### 4.2 SWOT 분석

**Strengths (강점)**
- 6단계 AI 추천 파이프라인 — 다차원 스코어링, 서사적 리포트, E2E 테스트 자동화

- 포괄적 관리자 시스템 (대시보드, 모더레이션, 통계)
- 프로덕션 수준 세션 관리 + 파일 업로드 3중 검증

**Weaknesses (약점)**

- 구조화 로깅/모니터링 미도입
- CI/CD 없이 수동 배포

**Opportunities (기회)**
- 자동완성 검색 도입으로 UX 대폭 개선 가능
- 비회원 AI 추천 체험으로 전환율 향상 가능

**Threats (위협)**

- AI 파이프라인 Workers 30초 한도 근접 → 복잡 케이스에서 타임아웃
- OpenAI 단일 의존 → API 장애/가격 변동 리스크

### 4.3 경쟁 포지셔닝

CareerWiki의 핵심 차별점은 **AI 추천 파이프라인의 깊이**입니다. 일반적인 진로 정보 사이트가 정적 데이터 제공에 그치는 반면, CareerWiki는 5축 프로필 수집 → LLM 심층 인터뷰 → Multi-Query 벡터 검색 → TAG 필터 → 맞춤 스코어링 → 서사적 리포트 생성까지 **6단계 파이프라인**을 완성했습니다. 여기에 위키 편집 시스템으로 커뮤니티 기여를 수용하는 **하이브리드 모델**은 경쟁 우위가 될 수 있습니다.

현재 기능 완성도(77.4%)와 AI 추천 품질은 충분히 경쟁력이 있으나, **보안/운영 안정화** 없이는 스케일링 단계에서 심각한 위험에 직면할 수 있습니다.
