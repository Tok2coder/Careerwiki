**1.3 대학 주요 교과목 리스트** (`mainSubject`)
- 위치: 전공 상세 페이지 중간
- 표시 내용:
  - 교과목명 (예: "민법총칙")
  - 교과목 설명
- UI 구성: 리스트 형태

**1.4 졸업 후 진출분야** (`enterField`)
- 위치: 전공 상세 페이지 하단
- 표시 내용:
  - 진출분야명 (예: "기업 및 산업체")
  - 진출분야 설명
- UI 구성: 섹션별로 구분

**1.5 통계 차트 시각화** (`chartData`)
- 위치: 전공 상세 페이지 중간/하단
- 표시할 통계:
  - 지원자 추이 (`applicant`)
  - 성별 분포 (`gender`)
  - 취업률 (`employment_rate`)
  - 분야별 분포 (`field`)
  - 평균 연봉 (`avg_salary`)
  - 만족도 (`satisfaction`)
  - 졸업 후 진로 (`after_graduation`)
- 차트 라이브러리: Chart.js 또는 D3.js
- UI 구성: 탭 또는 섹션별로 구분
