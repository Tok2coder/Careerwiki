## 📊 현재 템플릿에서 **사용 중인** 필드

### `src/templates/unifiedMajorDetail.ts`에서 사용 중:

#### 기본 정보
- ✅ `id` - 전공 고유 ID
- ✅ `name` - 전공명
- ✅ `sources` - 데이터 출처 (careernet, goyong24)
- ✅ `categoryName` - 계열/분야
- ✅ `categoryId` - 계열 ID

#### 설명 텍스트
- ✅ `summary` - 전공 개요
- ✅ `aptitude` - 이 전공에 어울리는 사람
- ✅ `whatStudy` - 무엇을 배우나요?
- ✅ `howPrepare` - 어떻게 준비하나요?

#### 진로 정보
- ✅ `jobProspect` - 진로 전망
- ✅ `relatedJobs` - 관련 직업 (배열)
- ✅ `employmentRate` - 취업률
- ✅ `salaryAfterGraduation` - 졸업 후 평균 연봉

#### 학습 정보
- ✅ `mainSubjects` - 주요 교과목 (배열)
- ✅ `licenses` - 추천 자격증 (배열)

#### 대학 정보
- ✅ `universities` - 개설 대학 (배열)
- ✅ `recruitmentStatus` - 모집 현황 (배열)

#### 관련 정보
- ✅ `relatedMajors` - 추천 유사 전공 (배열)

## ❌ 템플릿에서 **안 쓰이는** 필드 (D1에 수집했지만 미사용)

### Phase 1 필드 (CareerNet 전용)

#### 1. 고교 관련 과목
```typescript
relateSubject?: string[]  // ❌ 관련 고교 교과목
```
**설명**: 고등학교에서 이 전공 준비를 위해 수강하면 좋은 교과목  
**예시**: ["수학I", "수학II", "미적분", "확률과 통계"]  
**활용 방안**: "고교 추천 교과목" 섹션 추가

---

#### 2. 진로 탐색 활동
```typescript
careerAct?: string[]  // ❌ 진로 탐색 활동 가이드
```
**설명**: 전공 탐색을 위한 추천 활동  
**예시**: ["SW 개발 동아리 활동", "코딩 대회 참가", "오픈소스 기여"]  
**활용 방안**: "진로 탐색 활동" 섹션 추가

---

#### 3. 졸업 후 진출 분야
```typescript
enterField?: string[]  // ❌ 졸업 후 진출 분야
```
**설명**: 졸업 후 취업/진학 가능한 분야  
**예시**: ["IT 기업", "금융회사", "공공기관", "대학원 진학"]  
**활용 방안**: "졸업 후 진출 분야" 섹션 추가

---

#### 4. 전공 특성
```typescript
property?: string  // ❌ 전공 특성
```
**설명**: 전공의 특성 및 특징 설명  
**활용 방안**: "전공 특성" 섹션 또는 개요 섹션에 추가

---

#### 5. 통계 차트 데이터
```typescript
chartData?: {
  year?: string
  count?: string
  [key: string]: any
}[]  // ❌ 연도별 통계 차트 데이터
```
**설명**: 연도별 학생 수, 졸업생 수 등 통계 데이터  
**활용 방안**: 차트로 시각화

---

#### 6. 성별 코드 / 학교급 / 적성 리스트
```typescript
genCD?: string         // ❌ 성별 코드
schClass?: string      // ❌ 학교급
lstMiddleAptd?: any[]  // ❌ 중학교 적성 리스트
lstHighAptd?: any[]    // ❌ 고등학교 적성 리스트
lstVals?: any[]        // ❌ 가치관 리스트
```
**설명**: 세부 메타데이터 및 적성/가치관 정보  
**활용 방안**: 
- `lstMiddleAptd`, `lstHighAptd` - "적성 프로필" 섹션
- `lstVals` - "가치관" 섹션
