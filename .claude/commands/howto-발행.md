"HowTo 발행" 스킬을 실행한다.

`.claude/skills/howto-publish.md` 파일을 읽고, 그 안에 정의된 전체 워크플로우를 처음부터 끝까지 자율적으로 실행한다.

## 실행 순서

1. `.claude/skills/howto-publish.md`를 읽어 전체 워크플로우를 확인한다.
2. 기존에 발행된 HowTo 목록을 확인하여 중복되지 않는 주제를 선정한다.
3. 스킬 파일에 정의된 순서대로 실행한다:
   - 주제 선정 & 리서치 (웹 검색)
   - Tiptap JSON 콘텐츠 작성 (스킬 파일의 포맷 규칙 준수)
   - 썸네일 이미지 생성 (evolink API, HowTo 전용 slug)
   - 관련 직업/전공 매핑 (name 필드 필수)
   - publish-direct API 호출
   - 기술 검증 (체크리스트 전항목)
   - 내용 검증 (근거 기반 팩트체크)
4. 검증 결과를 표로 정리하여 보고한다.

## 주의사항
- 스킬 파일의 "절대 하지 말아야 할 것" 섹션을 반드시 준수한다.
- 세션 토큰이 없으면 유저에게 요청한다.
- 주제 선정 시 유저에게 묻지 않고 자율적으로 선택한다.
