## ETL 실행 명령어

```bash
# 1. 특정 직업들만 ETL 병합 실행
npm run etl:merge-jobs -- "가상현실전문가" "간호사" "간호조무사" "가수" "경제학연구원" "공업기계설치 및 정비원" "간병인" "간판제작원" "건축사" "3D프린팅모델러" "가사관리사" "감사사무원" "3D지도개발자" "3D프린터설치정비원" "3D프린팅운영기사" "GIS전문가" "IT컨설턴트" "가구제조,수리원" "IT기술지원전문가" "IT테스터 및 IT QA전문가" "UX·UI디자이너"
```

## 다음 단계

1. **ETL 실행**: 위 명령어로 21개 직업 병합
2. **템플릿 버전 업데이트**: `src/constants/template-versions.ts`에서 `JOB` 버전 증가
3. **서버 재시작**: `pm2 restart ecosystem.config.cjs`
4. **데이터 병합 로직 문서 작성**: 각 케이스별로 어떤 필드를 어떻게 표시할지 결정
5. **ETL `mergeJobProfiles.ts` 업데이트**: 문서화한 로직을 ETL에 반영
6. **템플릿 스타일 개선**: `unifiedJobDetail.ts`에서 프레젠테이션 로직만 유지
